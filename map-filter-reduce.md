
## 1️⃣ map →「一對一轉換」

> **用途：把陣列每個元素變成另一個元素**  
> **輸出長度 = 輸入長度**

    const nums = [1, 2, 3] 
    const doubled = nums.map(n => n * 2) // [2, 4, 6] 

💡 心法：

> 「我想要每個東西都變一個新東西」

----------

## 2️⃣ filter →「篩選」

> **用途：挑出符合條件的元素**  
> **輸出長度 ≤ 輸入長度**

    const nums = [1, 2, 3, 4] 
    const even = nums.filter(n => n % 2 === 0) // [2, 4]

💡 心法：

> 「我只要符合條件的那一部分」

----------

## 3️⃣ reduce →「合併 / 累加」

> **用途：把整個陣列縮減成一個值（或物件 / 陣列）**

    const nums = [1, 2, 3, 4] 
    const sum = nums.reduce((acc, n) => acc + n, 0) // 10 

💡 心法：

> 「一個一個拿來累積到結果」
