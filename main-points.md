
### 1️⃣ JavaScript（一定要到「不怕」的程度）

重點不是語法全背，而是**真的寫過**：

-   `let / const`
    
-   `array / object` 操作
    
-   `map / filter / reduce`
    
-   `async / await`
    
-   `fetch / axios`
    
-   ES module（`import / export`）
    

👉 面試會直接問，也會直接考你改 code

----------

### 2️⃣ React（超級關鍵）

Next.js 本質就是 **React 框架**

你至少要熟到：

-   Function Component
    
-   `useState`
    
-   `useEffect`
    
-   props / children
    
-   條件 render
    
-   list render（`map`）
    
-   表單處理（input、submit）
    

> ❗不懂 React，就算會 Next.js 也很難上崗

----------

## 二、Next.js 上崗核心（這一段是關鍵）

### 3️⃣ Next.js 基礎

一定要會這些（公司每天都在用）：

-   `app/` vs `pages/`（現在主流是 **app router**）
    
-   Routing（`page.tsx`、`layout.tsx`）
    
-   `Link`、`useRouter`
    
-   Server Component vs Client Component（**超愛考**）
    
-   `use client` 什麼時候要用
    

----------

### 4️⃣ Data Fetching（Next.js 的靈魂）

這一段會直接決定你是不是「初級 or 可用」

必會：

-   `fetch()`（server side）
    
-   SSR / SSG / ISR 概念
    
-   `cache: 'no-store'`
    
-   `revalidate`
    
-   loading / error handling
    

👉 面試常問：

> 為什麼這個頁面要用 SSR？

----------

### 5️⃣ API Route（前後端接起來）

你要能：

-   寫 `/api/*/route.ts`
    
-   處理 GET / POST
    
-   接 DB or mock data
    
-   回傳 JSON
    
-   驗證 request body
    

這一段 = **全端入門門票**

----------

## 三、實戰必備（很多人卡在這）

### 6️⃣ 登入 / 權限（一定會遇到）

不用一開始就很高級，但要懂：

-   cookie / token 概念
    
-   middleware（保護頁面）
    
-   登入後 redirect
    
-   logout flow（你最近就在寫這個 👍）
    

----------

### 7️⃣ Styling（至少會一種）

公司不在乎你用哪個，在乎你**能不能做 UI**

常見組合：

-   Tailwind CSS（🔥超多公司）
    
-   CSS Module
    
-   基本 responsive
    

----------

## 四、加分但很重要（讓你不像新手）

### 8️⃣ Git + 基本部署

至少要會：

-   `git clone / commit / push`
    
-   PR 基本流程
    
-   知道怎麼 deploy（Vercel / Docker）
